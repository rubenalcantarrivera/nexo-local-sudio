# Links para enviar

URL pública:

```txt
PUBLIC_URL=https://YOUR-VERCEL-URL.vercel.app
```

Reemplaza `PUBLIC_URL` con la URL real de Vercel cuando el despliegue del commit actual esté listo.

## Links principales

Homepage:

```txt
PUBLIC_URL/
```

Galería de ejemplos:

```txt
PUBLIC_URL/demos
```

Ejemplo dental:

```txt
PUBLIC_URL/demos/dental
```
