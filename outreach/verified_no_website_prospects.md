# Prospectos verificados sin sitio web claro - Nexo Local Studio

Última revisión: 2026-06-16

## 1. Summary

Se construyó una lista de 30 negocios locales reales en México con reputación pública, ubicación comercial fuerte y problema web verificado. Todos los prospectos incluidos tienen un estado de sitio compatible con el filtro: **No website on Google Maps / Places** u **Only Instagram/Facebook found**.

También se excluyeron 22 candidatos preliminares porque al abrir el detalle de Google Maps apareció un dominio oficial o sitio web propio, por ejemplo dominios de clínicas, ópticas, spas o restaurantes.

## 2. Method used

- Se buscó por nicho y zona en Google Maps usando consultas como dentista Polanco, clínica estética Juriquilla, veterinaria Providencia, fisioterapia Narvarte, etc.
- Se abrió el detalle del negocio en la vista pública de Google Maps para verificar rating, conteo de reseñas, categoría, dirección, teléfono y presencia/ausencia de dominio oficial.
- Se descartaron negocios con dominio oficial visible en Maps, cadenas nacionales evidentes o perfiles que quedaran dudosos.
- No se agregaron correos inventados ni personas de contacto inferidas. Cuando el nombre profesional aparece en el nombre público del negocio, se usó como contacto; en caso contrario, se marcó como **No público**.

Limitación: no se usó Google Places API porque no había llave API disponible en el entorno. La verificación se hizo con información pública visible en Google Maps y búsquedas exactas ligeras. Antes de contactar, conviene abrir de nuevo el perfil para confirmar que no hayan actualizado el sitio.

## 5. Notes on website verification

- **No website on Google Maps / Places:** el perfil público abierto en Maps no mostró dominio oficial ni enlace a sitio propio.
- **Only Instagram/Facebook found:** el perfil mostró presencia social/directorio, pero no dominio oficial propio.
- Los candidatos con dominios visibles como sitios de clínica, óptica, spa o restaurante fueron excluidos de la lista final.
- Las URLs de Google Maps en el CSV son enlaces de búsqueda exacta al negocio, útiles para reabrir y validar el perfil antes del primer contacto.
